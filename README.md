Feedbin Guide
-------------

### Keyboard Shortcuts

| Key                     | Action                             |
|-------------------------|------------------------------------|
| up, down, left, right   | Navigate through feeds and entries |
| space                   | Navigate through unread items      |
| s                       | Star entry                         |
| m                       | Toggle read/unread                 |
| g then a                | Go to all                          |
| g then s                | Go to starred                      |
| shift a                 | Mark all as read                   |
| a                       | Add subscription                   |
| escape                  | Unfocus subscription field         |
